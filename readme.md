# Blackjack

A simple blackjack game developed using Pygame and a Model-View-Controller Architecture.

![ui](promoimg.jpg)


### Installation (Windows)

```
git clone https://github.com/confy/blackjack.git
cd blackjack
py -m venv venv
.\venv\Scripts\activate
pip install pygame
py -m blackjack
```
